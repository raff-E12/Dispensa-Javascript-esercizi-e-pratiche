Le funzioni

La funzione è una porzione di codice che può essere eseguita quando la richiamiamo all'interno del codice. Ma prima di poterla richiamare ne dobbiamo creare una. Molto spesso è necessario eseguire azioni simili in diverse parti di uno script. Ad esempio, potremmo voler mostrare un messaggio quando un utente effettua il login o il logout, oppure per altri motivi. Le funzioni permettono di riutilizzare il codice, evitando ripetizioni. Abbiamo già visto esempi di funzioni integrate come alert(message), prompt(message, default) e confirm(question).

Dichiarazione di una funzione, avviene attraverso la parola function e subito dopo il nome che vogliamo assegnare alla funzione.


					function mostraMessaggio() {
						alert( 'Good morning!' )
					  }
					  mostraMessaggio() //richiamiamo la funzione per farla partire

esempio di funzione con variabili


					let nome = 'Giorgio'; //variabile esterna alla funzione,
					 può essere usata all'interno della funzione
					  e può essere anche modificata dalla funzione.

					function mostraMessaggio() {
					let messaggio = 'Ciao, ' + nome; // variabile interna, può essere usata
					solo all'interno della funzione
					alert(messaggio);
					}

					mostraMessaggio(); // Ciao Giorgio
					console.log(messaggio)

Alcune funzioni interne per l'output

document.write()

Scrive nell'html il valore contenuto tra le parentesi, nel punto in cui si trova.

innerHTML

Scrive o sovrascrive un valore all'interno dell'elemento html selezionato

testi da sovrascrivere

alert() o window.alert()

Crea una finestra modale o anche detta popup

console.log()

Scrive all'interno della console javascript del tuo browser, utilizzato spesso per il debugging del codice
