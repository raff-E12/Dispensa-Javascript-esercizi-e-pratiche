Sintassi Jx, variabili e operatori

La sintassi

Per scrivere uno script bisogna conoscere l'insieme di regole che compongono la sintassi di Jx.

document.write('Testo da visualizzare');

document è un oggetto e rappresenta l'intera pagina web

"." il punto rappresenta l'operatore membro che ci permette di accedere ai vari metodi e proprietà dell'oggetto in questione.

"write('Testo da visualizzare')" rappresenta il metodo dell'oggetto document richiamato per scrivere un nuovo contenuto sulla pagina, all'interno delle parentesi troviamo i parametri del metodo, in questo caso la stringa da scrivere sulla pagina web.

Per chiudere ogni istruzione si utilizza il ;

Variabili

Le variabili sono dei contenitori dove uno script può conservare temporaneamente informazioni che servono poi per svolgere il proprio lavoro.

Il nome delle variabili può iniziare con $, _ o lettera, non può iniziare con un numero. Non sono ammessi i seguenti caratteri "-" e "."

Dichiarazione di una variabile

Es. let x;

let è la parola chiave per dichiarare la variabile.

x è il nome assegnato a questa variabile.

Assegnare un valore alle variabili

x = 3;

"=" è l'operatore di assegnazione.

Adesso quando richiamiamo la variabile all'interno di uno script x avrà come valore 3.

Vedi il codice per l'esempio

Valori contenuti da una variabile possono essere

  • Valori stringa --> "ciao" oppure 'ciao'
  • Valori numerici interi e decimali --> 10 , 10.25
  • Valori boleani --> true / false valore astratto che indica uno stato di vero o falso, es. per verificare se una condizione si verifica o meno.

Una variabile una volte dichiarata ha un valore indefinito che cambia con l'assegnazione. (x='ciao' diventa automaticamente una varibiale con valore stringa). La variabile una volta dichiarata ed assegnata, può essere riassegnata.

Le constanti

Per dichiarare una costante (il valore non cambia una volta assegnato), dobbiamo utilizzare const invece di let. Una volta assegnate non possono cambiare valore.


const nome = 'Giorgio'; nome= 'Mario'; // errore non può essere riassegnata

Tipi di dato

Un valore in JavaScript è sempre associato a un tipo specifico, come ad esempio string o number. Una variabile in JavaScript può contenere qualsiasi tipo di dato.

Number

Numeri vanno senza apici e sono di due tipi interi e decimali.

let x=3; x=3.14;

String

La stringa è un tipo di dato che va tra apici.

let str='ciaooo'; console.log(str)

Boolean

Un valore di tipo boolean ha solo due valori: true e false. Questo tipo viene usato principalmente per memorizzare valori “si/no”: true significa “Si, corretto”, e false significa “No, scorretto”, serve per verificare le condizioni logiche.

let consensoCheck = true; // il campo è spuntato consensoCheck = false; // il campo non è spuntato

Null e Undefined

Valore speciale usato solo per indicare che il valore è nullo o vuoto

let y = null; console.log(y)

Undefined significa 'valore non assegnato'. Esempio,se una variabile viene dichiarata, ma non assegnata, il suo valore è undefined

let z; console.log(z)

Typeof

L’operatore typeof ritorna di che tipo è il valore. Comodo per eseguire controlli rapidi sul tipo del valore delle variabili.

let a=3 console.log(typeof a)

Operatori

Operatori matematici

+ - / * % ++ --

  • + operatore di addizione
  • - operatore di sottrazione
  • / operatore di divisione
  • * operatore di moltiplicazione
  • % operatore di resto
  • ++ operatore di incremento
  • -- operatore di decremento

Operatori di comparazione

resituiscono un valore di tipo booleano (true o false)

  • == uguale a
  • === uguale valore e tipo
  • != non uguale
  • !== non uguale valore o non uguale di tipo
  • > più grande di
  • < più piccolo di
  • >= più grande di o uguale
  • <= più piccolo di o uguale

Operatori logici && || !

  • && operatore di congiunzione, trova il primo valore falso, altrimenti va avanti tra le condizioni
  • || operatore "o" (or) oppure
  • ! operatore di negazione
