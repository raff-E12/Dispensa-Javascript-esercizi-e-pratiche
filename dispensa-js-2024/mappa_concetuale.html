Appunti di JavaScript

  • Variabili

    Le variabili in JavaScript sono utilizzate per memorizzare dati. Puoi usare var, let, o const.

    Esempio:

    var nome = 'Mario'; // Variabile globale o di funzione
let eta = 30; // Variabile a blocco
const città = 'Roma'; // Variabile costante

    Proprietà e Metodi:

    Le variabili di per sé non hanno proprietà o metodi, ma i valori che contengono possono essere di diversi tipi con i loro metodi.

  • Tipi di Dati

    JavaScript supporta diversi tipi di dati come stringhe, numeri, booleani, oggetti, array, e null.

    Esempio:

    let nome = 'Mario'; // Stringa
let eta = 30; // Numero
let isStudente = true; // Booleano
let persona = { nome: 'Mario', eta: 30 }; // Oggetto
let numeri = [1, 2, 3, 4, 5]; // Array
let valoreNull = null; // Null

    Proprietà e Metodi:

    Ogni tipo di dato ha proprie proprietà e metodi. Ad esempio, le stringhe hanno metodi come length e toUpperCase().

    Stringhe

    let testo = 'Ciao, mondo!';
console.log(testo.length); // Output: 12
console.log(testo.toUpperCase()); // Output: 'CIAO, MONDO!'
console.log(testo.substring(0, 4)); // Output: 'Ciao'

    Numeri

    let numero = 123.456;
console.log(numero.toFixed(2)); // Output: '123.46'
console.log(Number.isInteger(numero)); // Output: false

    Array

    let numeri = [1, 2, 3, 4, 5];
console.log(numeri.length); // Output: 5
numeri.push(6);
console.log(numeri); // Output: [1, 2, 3, 4, 5, 6]
let nuoviNumeri = numeri.map(n => n * 2);
console.log(nuoviNumeri); // Output: [2, 4, 6, 8, 10, 12]

  • Funzioni

    Le funzioni sono blocchi di codice che possono essere definiti una volta e eseguiti molte volte.

    Esempio:

    function saluta(nome) {
    return 'Ciao ' + nome + '!';
}

console.log(saluta('Mario')); // Output: Ciao Mario!

    Proprietà e Metodi:

    Le funzioni possono avere proprietà come name e length. Possono anche essere assegnate a variabili o passate come argomenti.

    Proprietà delle Funzioni

    function esempio(a, b) {
    return a + b;
}
console.log(esempio.name); // Output: 'esempio'
console.log(esempio.length); // Output: 2

    Funzioni Anonime e Arrow Functions

    let somma = function(a, b) {
    return a + b;
};
console.log(somma(1, 2)); // Output: 3

let moltiplica = (a, b) => a * b;
console.log(moltiplica(2, 3)); // Output: 6

  • Oggetti

    Gli oggetti sono collezioni di proprietà e metodi. Sono utilizzati per rappresentare entità più complesse.

    Esempio:

    let persona = {
    nome: 'Mario',
    eta: 30,
    saluta: function() {
        return 'Ciao, sono ' + this.nome;
    }
};

console.log(persona.saluta()); // Output: Ciao, sono Mario

    Proprietà e Metodi:

    Gli oggetti possono avere proprietà (chiave: valore) e metodi (funzioni). Si possono accedere usando la notazione del punto o delle parentesi quadre.

    Proprietà degli Oggetti

    let libro = {
    titolo: 'Il Nome della Rosa',
    autore: 'Umberto Eco',
    anno: 1980
};
console.log(libro.titolo); // Output: 'Il Nome della Rosa'
console.log(libro['autore']); // Output: 'Umberto Eco'

    Metodi degli Oggetti

    let calcolatrice = {
    somma: function(a, b) {
        return a + b;
    },
    sottrai: function(a, b) {
        return a - b;
    }
};
console.log(calcolatrice.somma(5, 3)); // Output: 8
console.log(calcolatrice.sottrai(5, 3)); // Output: 2

  • Array

    Gli array sono usati per memorizzare liste ordinate di valori.

    Esempio:

    let numeri = [1, 2, 3, 4, 5];

console.log(numeri[0]); // Output: 1
numeri.push(6); // Aggiunge un elemento alla fine dell'array
console.log(numeri); // Output: [1, 2, 3, 4, 5, 6]

    Proprietà e Metodi:

    Gli array hanno proprietà come length e metodi come push(), pop(), shift(), unshift(), map(), filter(), e molti altri.

  • Modello DOM

    Il Document Object Model (DOM) è una rappresentazione strutturata del documento HTML. Permette di manipolare il contenuto e la struttura della pagina web.

    Esempio:

    <div id="contenitore">
            <p id="paragrafo">Testo originale</p>
        </div>
        
        <script>
        document.getElementById('paragrafo').textContent = 'Testo modificato';
        document.getElementById('contenitore').style.backgroundColor = 'lightblue';
        </script>

    Proprietà e Metodi:

    Il DOM fornisce metodi e proprietà per interagire con gli elementi HTML.

    Proprietà del DOM

    let paragrafo = document.getElementById('paragrafo');
        console.log(paragrafo.textContent); // Output: 'Testo modificato'
        console.log(paragrafo.innerHTML); // Output: 'Testo modificato'
        console.log(paragrafo.className); // Output: (classe dell'elemento, se presente)
        console.log(paragrafo.id); // Output: 'paragrafo'

    Metodi del DOM

    let nuovoParagrafo = document.createElement('p');
        nuovoParagrafo.textContent = 'Nuovo paragrafo';
        document.body.appendChild(nuovoParagrafo); // Aggiunge il nuovo paragrafo al body
        
        let elemento = document.getElementById('contenitore');
        elemento.removeChild(nuovoParagrafo); // Rimuove il nuovo paragrafo dal contenitore
        
        let paragrafoEsistente = document.getElementById('paragrafo');
        paragrafoEsistente.replaceWith(nuovoParagrafo); // Sostituisce il paragrafo esistente con il nuovo

    Manipolazione del DOM

    let contenitore = document.getElementById('contenitore');
        let nuovoElemento = document.createElement('span');
        nuovoElemento.textContent = 'Elemento aggiunto';
        contenitore.appendChild(nuovoElemento); // Aggiunge un elemento all'interno del contenitore
        
        let paragrafo = document.getElementById('paragrafo');
        paragrafo.style.color = 'red'; // Cambia il colore del testo del paragrafo
        
        let paragrafo2 = document.createElement('p');
        paragrafo2.textContent = 'Un altro paragrafo';
        document.querySelector('div').insertBefore(paragrafo2, document.querySelector('p')); // Inserisce un nuovo paragrafo prima del primo paragrafo esistente

  • Eventi

    Gli eventi sono azioni che accadono nel browser, come il click di un bottone o il caricamento di una pagina.

    Esempio:

    <button onclick="mostraMessaggio()">Cliccami</button>

<script>
function mostraMessaggio() {
alert('Bottone cliccato!');
}
</script>

    Proprietà e Metodi:

    Gli eventi possono essere gestiti con proprietà come type e target. I metodi comuni includono addEventListener() per aggiungere gestori di eventi.

  • Responsive Design con MediaQueryList

    Il metodo matchMedia() di JavaScript permette di verificare se la finestra del browser corrisponde a una determinata media query, utile per creare design responsive.

    Esempio:

    const mediaQuery = window.matchMedia('(max-width: 1200px)');
                    
                    function handleResize(e) {
                        if (e.matches) {
                            console.log('La finestra è sotto i 1200px di larghezza.');
                            // Aggiungi qui il codice per il design mobile
                        } else {
                            console.log('La finestra è sopra i 1200px di larghezza.');
                            // Aggiungi qui il codice per il design desktop
                        }
                    }
                    
                    // Aggiunge un listener per i cambiamenti della media query
                    mediaQuery.addEventListener('change', handleResize);
                    
                    // Esegui la funzione subito per gestire lo stato iniziale
                    handleResize(mediaQuery);

    Proprietà e Metodi:

    L'oggetto MediaQueryList restituito da matchMedia() ha diverse proprietà e metodi per gestire le media query.

    Proprietà

    mediaQuery.matches // Ritorna true se la media query corrisponde alle condizioni attuali
                    mediaQuery.media // Contiene la stringa della media query (es. '(max-width: 1200px)')

    Metodi

    mediaQuery.addEventListener('change', listenerFunction); // Aggiunge un gestore di eventi per i cambiamenti della media query
                    mediaQuery.removeEventListener('change', listenerFunction); // Rimuove un gestore di eventi precedentemente aggiunto
