Introduzione a JavaScript

JavaScript è un linguaggio di programmazione leggero e versatile, essenziale per lo sviluppo web moderno. Permette di creare interazioni dinamiche e ricche all'interno delle pagine web, e può essere eseguito sia sul lato client (nel browser) che sul lato server (con Node.js).

JavaScript è nato nel 1995 per aggiungere comportamenti dinamici ai siti web e da allora è diventato uno dei pilastri del web development.

Elementi Fondamentali di JavaScript

Variabili

Le variabili sono utilizzate per memorizzare dati che possono essere utilizzati e manipolati nel tuo programma. In JavaScript, puoi dichiarare variabili utilizzando var, let e const:

  • var: Dichiarazione di variabili con ambito di funzione. Può essere riassegnato e ridefinito.
  • let: Dichiarazione di variabili con ambito di blocco. Può essere riassegnato ma non ridefinito all'interno dello stesso blocco.
  • const: Dichiarazione di variabili con ambito di blocco che non possono essere riassegnate. Tuttavia, se la variabile contiene un oggetto, le proprietà di quell'oggetto possono essere modificate.
                
                    var nome = 'Mario';
                    let eta = 25;
                    const paese = 'Italia';

Tipi di Dati

JavaScript supporta diversi tipi di dati, tra cui:

  • Numeri: Utilizzati per rappresentare sia interi che numeri a virgola mobile.
  • Stringhe: Sequenze di caratteri racchiuse tra virgolette singole o doppie.
  • Booleani: Valori logici che possono essere true o false.
  • Oggetti: Collezioni di proprietà e metodi.
  • Funzioni: Blocchi di codice riutilizzabili che possono essere chiamati con un nome.
                
                    let numero = 42;
                    let testo = 'Ciao, mondo!';
                    let veroFalso = true;
                    let oggetto = { nome: 'Mario', eta: 25 };
                    let funzione = function() { return 'Ciao!'; };

Funzioni

Le funzioni in JavaScript sono blocchi di codice riutilizzabili che possono essere chiamati per eseguire una determinata operazione. Ci sono diversi modi per dichiarare funzioni:

Funzioni Dichiarate

Le funzioni dichiarate vengono definite utilizzando la parola chiave function, seguita dal nome della funzione, dalle parentesi tonde e dal corpo della funzione racchiuso tra parentesi graffe:

                
                    function saluta() {
                        console.log('Ciao!');
                    }
                    saluta(); // Chiamata della funzione

Funzioni Espresse

Le funzioni espresse vengono assegnate a una variabile. Queste funzioni non hanno un nome proprio e vengono invocate utilizzando il nome della variabile:

                
                    const saluta = function() {
                        console.log('Ciao!');
                    };
                    saluta(); // Chiamata della funzione

Funzioni Freccia

Le funzioni freccia sono una sintassi più concisa per dichiarare funzioni. Utilizzano l'operatore freccia (=>) e non legano il proprio this:

                
                    const saluta = () => {
                        console.log('Ciao!');
                    };
                    saluta(); // Chiamata della funzione

Manipolazione del DOM

Il DOM (Document Object Model) rappresenta la struttura di un documento HTML o XML. JavaScript può essere utilizzato per manipolare il DOM, permettendo di aggiornare dinamicamente il contenuto e la struttura delle pagine web.

Selezione degli Elementi

Per selezionare elementi del DOM, puoi utilizzare metodi come getElementById, getElementsByClassName, e querySelectorAll:

                
                    const paragrafo = document.getElementById('mioParagrafo');
                    const paragrafi = document.getElementsByClassName('paragrafo');
                    const elementi = document.querySelectorAll('.classe');

Modifica del Contenuto

Una volta selezionato un elemento, puoi modificarne il contenuto utilizzando proprietà come textContent:

                
                    const paragrafo = document.getElementById('mioParagrafo');
                    paragrafo.textContent = 'Nuovo contenuto!';

Questo è un paragrafo.

Gestione degli Eventi

JavaScript può rispondere a vari eventi dell'utente, come clic, passaggi del mouse e pressioni di tasti. Questo permette di creare interfacce utente interattive.

                
                    const bottone = document.getElementById('mioBottone');
                    bottone.addEventListener('click', function() {
                        alert('Bottone cliccato!');
                    });

Comunicazione con il Server

JavaScript può inviare e ricevere dati dal server senza ricaricare la pagina usando AJAX (Asynchronous JavaScript and XML) e la Fetch API. Questo permette di creare esperienze utente più dinamiche e responsive.

AJAX con XMLHttpRequest

AJAX è una tecnica per creare applicazioni web interattive. Con AJAX, puoi caricare dati in background e visualizzare questi dati nella pagina web senza ricaricare l'intera pagina. Ecco un esempio di richiesta AJAX utilizzando XMLHttpRequest:

                
                    const xhr = new XMLHttpRequest();
                    xhr.open('GET', 'https://jsonplaceholder.typicode.com/posts');
                    xhr.onload = function() {
                        if (xhr.status === 200) {
                            console.log(JSON.parse(xhr.responseText));
                        }
                    };
                    xhr.send();

Fetch API

La Fetch API fornisce un'interfaccia moderna per effettuare richieste HTTP. È più facile da usare rispetto a XMLHttpRequest e supporta promesse, rendendo il codice più leggibile e gestibile:

                
                    fetch('https://jsonplaceholder.typicode.com/posts')
                        .then(response => response.json())
                        .then(data => console.log(data))
                        .catch(error => console.error('Errore:', error));

DOMTokenList

L'interfaccia DOMTokenList in JavaScript rappresenta un insieme di token separati da spazi. Questo è utilizzato principalmente per manipolare l'attributo class degli elementi HTML, ma può essere utilizzato anche per altri attributi come rel, aria-labelledby, ecc.

Gestione delle Classi sugli Elementi HTML: La proprietà classList di un elemento restituisce un DOMTokenList che permette di gestire le classi CSS associate a quell'elemento.

Metodi Principali di DOMTokenList:

  • add(token): Aggiunge il token specificato alla lista, se non è già presente.
  • remove(token): Rimuove il token specificato dalla lista.
  • toggle(token): Alterna la presenza del token nella lista. Se il token è presente, lo rimuove, altrimenti lo aggiunge.
  • contains(token): Restituisce true se il token specificato è presente nella lista, altrimenti false.
  • item(index): Restituisce il token all'indice specificato.

Conversione dei Dati e tipizzazione debole

Javascript a differenza degli altri linguaggi di programmazione è in grado di convertire ogni tipo di dato per completare ogni determinata operazione
come ad esempio '3' > 0 il risultato sarà 'true' per una semplicissima ragione, perchè visto che il confroto '3' lo converte in un numero del tutto reale da darà per forza come risultato 'true'.

tutto questo concetto si può riassumere in Tipizzazione Debole ovvero per ogni dato usato in una variabile che sia un booleano, un numero o una stringa lo converte automaticamente durante una condizione o un ciclo di qualsiasi tipo.

                
                    var btn = document.getElementById('vr');
                    var txt = document.getElementById('risult');
                    var calc = '3' > 0;
                    btn.onclick = function(){
                        txt.innerHTML = calc;
                    };
